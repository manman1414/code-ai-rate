# Privacy Policy

**Code AI Rate** — 最后更新：2026-06-28

## 概述

Code AI Rate 是一款完全在本地运行的 VS Code 扩展。我们不会在您的设备之外收集、存储或传输任何数据。

## 网络访问

- **零网络请求** — 扩展不发起任何 HTTP、WebSocket 或其他出站网络连接
- 所有文件读取、分析与报告生成均在您的本地机器上完成

## 遥测与追踪

- **零遥测** — 不使用任何分析、追踪或崩溃报告服务
- 不收集使用统计、用户标识或工作区元数据

## 数据处理

扩展仅在您主动触发扫描时读取工作区内的源代码文件。处理结果保存在 VS Code 进程内存中，或写入您通过导出功能自行选择的本地文件路径。扩展不会自动上传或同步任何内容。

## 第三方服务

扩展不依赖任何第三方 API 或云服务。

## 变更

如有隐私政策变更，将在 CHANGELOG 中注明并在后续版本中更新本文档。

## 联系

如有隐私相关问题，请通过项目仓库的 Issue 页面联系维护者。
